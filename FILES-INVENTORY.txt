DEPLOYMENT OPTIONS:
===================

Option 1: PowerShell Script (Recommended for Windows)
------
cd C:\path\to\landing-zone-demo
.\deploy.ps1 -AdminPassword "SecurePassword123!"

Option 2: Bash Script (Recommended for Linux/Mac)
------
cd /path/to/landing-zone-demo
./deploy.sh rg-landing-zone-demo eastus "SecurePassword123!"

Option 3: Azure CLI (Manual)
------
az deployment sub create \
  --location eastus \
  --template-file infra/main.bicep \
  --parameters infra/main.parameters.json \
  --parameters adminPassword="SecurePassword123!"

Option 4: Azure Portal
------
1. Open Azure Portal
2. Search for "Deploy a custom template"
3. Upload infra/main.bicep
4. Fill in parameters
5. Review and create
